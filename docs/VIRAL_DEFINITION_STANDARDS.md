# 🎯 YouTube Shorts & TikTok 爆款定义专业标准

## 📋 概述

基于行业最佳实践和数据分析，本文档定义了YouTube Shorts和TikTok平台的爆款内容评判标准。

---

## 📊 量化指标（硬性标准）

### 1. 播放量标准

#### TikTok
```
🔥 确定爆款：100万+ 播放
🌟 热门视频：50万+ 播放
⭐ 潜力视频：10万+ 播放
○  正常视频：1万+ 播放
```

#### YouTube Shorts
```
🔥 确定爆款：50万+ 播放
🌟 热门视频：20万+ 播放
⭐ 潜力视频：5万+ 播放
○  正常视频：5千+ 播放
```

**为什么有差异？**
- TikTok用户基数更大，传播速度更快
- YouTube Shorts搜索流量占比更高，长尾效应更明显

---

### 2. 互动率标准

#### 点赞率（点赞数 / 播放量）
```
💯 极优：10%+ 
✅ 优秀：5%+
○  正常：2%+
⚠️  较差：1%+
```

#### 评论率（评论数 / 播放量）
```
💯 极优：2%+
✅ 优秀：1%+
○  正常：0.5%+
⚠️  较差：0.2%+
```

#### 分享率（分享数 / 播放量）⭐ **关键传播指标**
```
💯 极优：3%+
✅ 优秀：1.5%+
○  正常：1%+
⚠️  较差：0.5%+
```

**为什么分享最重要？**
- 分享 = 主动传播 = 二次爆发
- 平台算法更青睐被分享的内容
- 分享意味着内容有足够的价值让用户愿意推荐

#### 综合互动率
```
💯 极优：15%+  (点赞+评论+分享)
✅ 优秀：8%+
○  正常：4%+
⚠️  较差：2%+
```

---

### 3. 完播率标准（如果可获取）

```
💯 极优：70%+ 说明内容吸引力极强
✅ 优秀：60%+ 内容质量很好
○  正常：50%+ 基本合格
⚠️  较差：40%+ 需要优化
```

**算法逻辑：**
- 完播率 > 60% → 平台持续推荐
- 完播率 < 40% → 限制流量
- 重复观看 → 算法加权推荐

---

### 4. 时间窗口标准

#### 爆发期判断
```
⚡ 快速爆发：前 6小时 内快速增长
🔥 持续爆发：前 24小时 内持续增长
📈 稳定增长：前 72小时 内持续获得推荐
```

#### 二次爆发
```
🌊 最佳窗口：7-14天 后再次被推荐
💡 原因：算法重新评估内容价值，推送给新用户群
```

#### 内容新鲜度
```
🔴 最新：1天内   （最高权重）
🟠 新鲜：3天内   （高权重）
🟡 正常：7天内   （正常权重）
🟢 较旧：14天内  （降低权重）
```

## 🎭 内容特征（软性标准）

### 1. 开头设计（前3秒）

**必须包含至少一个元素：**

✅ **视觉冲击**
- 鲜艳色彩、快速剪辑、特效
- 示例："画面突然变色" "快速切换场景"

✅ **悬念设置**
- 制造问题、抛出疑问
- 示例："你知道吗..." "居然有人..."

✅ **冲突引入**
- 对比、矛盾、意外
- 示例："富人vs穷人" "期望vs现实"

✅ **好奇心激发**
- 未知、秘密、揭秘
- 示例："千万别这样做" "背后的真相"

### 2. 钩子文案特征

高分文案模式：

| 模式 | 示例 | 评分加成 |
|------|------|----------|
| 你知道吗 | "你知道为什么..." | +15分 |
| 惊讶反转 | "居然/竟然..." | +18分 |
| 禁止警告 | "千万别/不要..." | +16分 |
| 方法指导 | "如何/怎么..." | +14分 |
| 数字列表 | "5个技巧..." | +12分 |
| 秘密揭秘 | "XXX的秘密" | +17分 |

### 3. 情绪共鸣触发

**强烈情绪类型及效果：**

```
😂 搞笑 (1.2x加权) - 最易传播
😢 感动 (1.3x加权) - 深度共鸣
😲 惊讶 (1.15x加权) - 记忆深刻
😠 愤怒 (1.1x加权) - 引发讨论
💪 励志 (1.25x加权) - 激发行动
```

### 4. 趋势把握

✅ **热门音乐/音效**
- 使用近7天热门音乐 → 1.3x加权
- 使用经典音乐 → 1.1x加权
- 使用原创音乐 → 1.15x加权

✅ **参与平台挑战**
- 参与活跃挑战 → 1.25x加权
- 参与过时挑战 → 1.05x加权

✅ **跟进时事热点**
- 紧跟热点（3天内）→ 最高权重
- 已有热度（7天内）→ 中等权重

## 📏 相对定义标准

### 1. 账号基础分层

爆款是**相对**的，需要考虑账号基础：

| 账号等级 | 粉丝数 | 相对标准（播放/粉丝比） |
|---------|--------|----------------------|
| 🌟 超大号 | 100万+ | 50% (50万播放 for 100万粉) |
| 🔥 大号 | 10万+ | 100% (10万播放 for 10万粉) |
| ⭐ 中号 | 1万+ | 300% (3万播放 for 1万粉) |
| ○ 小号 | 1千+ | 1000% (1万播放 for 1千粉) |
| • 微小号 | <1千 | 5000% (5万播放 for 1千粉) |

**实例：**
```
案例A：100万粉丝账号的10万播放
→ 相对表现：10% (10万/100万)
→ 判断：较差，未达标

案例B：1千粉丝账号的5万播放
→ 相对表现：5000% (5万/1千)
→ 判断：爆款！远超预期
```

## 🔍 综合判断算法

### 评分维度（总分100）

```typescript
爆款评分 = 
  播放量基础分 (30分) +
  互动率得分 (25分) +
    - 点赞率 (8分)
    - 评论率 (7分)
    - 分享率 (10分) ← 最重要
  +
  相对表现分 (15分) +
    - 基于粉丝数的相对表现
  +
  内容质量分 (20分) +
    - 开头设计 (7分)
    - 情绪共鸣 (6分)
    - 趋势把握 (7分)
  +
  时间新鲜度 (10分)
```

### 判断逻辑

```typescript
if (总分 >= 85) {
  return "🔥 确定爆款"
}
else if (总分 >= 70) {
  return "🌟 热门视频"
}
else if (总分 >= 55) {
  return "⭐ 潜力视频"
}
else {
  return "○ 普通视频"
}
```

### 置信度计算

```typescript
置信度 = min(总分, 100)

高置信度 (85+)  → 强烈推荐复制
中置信度 (70-84) → 可以参考
低置信度 (<70)  → 谨慎对待
```


### 批量筛选

```typescript
import { isViral } from '@/lib/viral-definition-standards';

const videos = [...]; // 爬取的视频列表

const viralVideos = videos
  .map(video => ({
    ...video,
    viralAnalysis: isViral(video, 'youtube_shorts'),
  }))
  .filter(v => v.viralAnalysis.isViral)
  .sort((a, b) => b.viralAnalysis.confidence - a.viralAnalysis.confidence);

console.log(`找到 ${viralVideos.length} 个爆款视频`);
```
