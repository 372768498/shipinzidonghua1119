## 设计概览

### 🎨 设计理念

**关键词**: 简洁、高效、专业

Jilo.ai的设计遵循以下核心原则：

1. **内容优先**: 减少视觉干扰，突出核心功能
2. **效率至上**: 减少用户的点击次数，最短路径完成任务
3. **数据可视**: 清晰展示视频数据和生成进度
4. **专业感**: 面向内容创作者和企业用户的专业工具

### 🎨 品牌色彩

```
主色调 (Primary)
- 色值: #6366f1 (Indigo)
- 用途: 主要按钮、链接、强调元素
- 寓意: 科技、AI、创新

辅助色 (Secondary)
- 色值: #f472b6 (Pink)
- 用途: 创意相关、视频元素
- 寓意: 创意、内容、视频

成功色 (Accent)
- 色值: #10b981 (Green)
- 用途: 成功状态、增长数据
- 寓意: 增长、成功、完成

中性色
- 背景: #ffffff (白色)
- 文字: #1e293b (深灰)
- 边框: #e2e8f0 (浅灰)
- 次要文字: #64748b (中灰)
```

### 📐 布局系统

**网格系统**: 12列栅格布局  
**断点**: 
- 桌面: ≥1280px
- 平板: 768px - 1279px
- 手机: <768px

**间距规范**:
- 超小: 4px
- 小: 8px
- 中: 16px
- 大: 24px
- 超大: 32px

**路径**: `/login`, `/register`  
**布局**: 左右分栏布局

│  │ Jilo.ai  │        │   [登录按钮]    │       │
│  │          │        │                 │       │
│  │ 让视频创作│        │   [Google登录]  │       │
│  │ 变得简单  │        │                 │       │

#### 左侧 - 品牌展示区 (400px宽)

**背景**: 品牌主色 (#6366f1)

**内容**:
- Logo图标: 🎬 (96px)
- 产品名称: "Jilo.ai" (48px, 加粗, 白色)
- Slogan: "让视频创作变得像呼吸一样简单" (20px, 白色, 透明度90%)

#### 右侧 - 登录表单区

**背景**: 白色

**内容元素**:

1. **页面标题**
   - 主标题: "欢迎回来" (36px, 加粗)
   - 副标题: "登录继续您的创作之旅" (16px, 次要文字色)
   - 间距: 标题下方32px

2. **邮箱输入框**
   - 标签: "邮箱地址" (14px, 次要文字色)
   - 输入框: 
     - 高度: 48px
     - 圆角: 8px
     - 背景: #f1f5f9
     - Placeholder: "your@email.com"
     - 获焦时: 蓝色边框

3. **密码输入框**
   - 标签: "密码" (14px, 次要文字色)
   - 输入框: 
     - 高度: 48px
     - 类型: password
     - 右侧: 眼睛图标（显示/隐藏密码）
   - 下方: "忘记密码？" 链接（右对齐）

4. **登录按钮**
   - 文字: "登录"
   - 尺寸: 100%宽 × 48px高
   - 背景: 主色 (#6366f1)
   - 文字: 白色, 16px
   - 悬停: 颜色加深10%
   - 点击: 显示loading动画

5. **分隔线**
   - "或" 居中，两侧灰线
   - 上下间距: 24px

6. **Google登录按钮**
   - 文字: "🔐 使用Google登录"
   - 尺寸: 100%宽 × 48px高
   - 背景: 白色
   - 边框: 1px 灰色 (#e2e8f0)
   - 悬停: 背景变为浅灰

7. **注册链接**
   - 文字: "还没有账号？立即注册"
   - 颜色: "立即注册"为主色
   - 位置: 底部居中
   - 间距: 顶部32px

#### 交互说明

1. **表单验证**:
   - 实时验证: 邮箱格式、密码长度
   - 错误提示: 输入框下方显示红色错误文字
   - 成功提示: 绿色勾号图标

2. **登录流程**:
   ```
   用户输入 → 前端验证 → 提交 → 后端验证 → 
   成功: 跳转Dashboard | 失败: 显示错误提示
   ```

3. **Google OAuth**:
   ```
   点击按钮 → 弹出Google授权窗口 → 授权成功 → 
   创建账号/登录 → 跳转Dashboard
   ```

#### 状态处理

| 状态 | 视觉表现 | 用户反馈 |
|------|----------|----------|
| **空闲** | 正常显示 | - |
| **填写中** | 输入框获焦高亮 | 实时验证提示 |
| **提交中** | 按钮显示loading | 禁用所有输入 |
| **成功** | 显示成功图标 | 1秒后跳转 |
| **失败** | 错误提示红色 | 允许重新输入 |

## 响应式设计

### 断点定义

```
桌面端 (Desktop): ≥ 1280px
平板端 (Tablet):  768px - 1279px
手机端 (Mobile):  < 768px
```

### 适配策略

#### 桌面端 (≥1280px)
- 完整显示所有功能
- 侧边栏展开
- 多列布局
- 悬停效果完整

#### 平板端 (768px - 1279px)
- 侧边栏可收起为图标
- 2列布局改为1列
- 表单和输入框保持合理宽度
- 触摸友好的按钮大小

#### 手机端 (<768px)
- 侧边栏改为底部导航
- 单列布局
- 全宽按钮
- 简化的导航栏
- 抽屉式菜单

### 移动端特殊优化

**Dashboard页面**:
- 统计卡片改为单列垂直排列
- 视频卡片改为列表样式
- 导航改为底部Tab栏

**生成页面**:
- 配置区和预览区改为上下布局
- 配置区可折叠
- 预览区占据主要空间

**发布页面**:
- 视频预览置顶
- 表单区域可滚动
- 发布按钮固定在底部
